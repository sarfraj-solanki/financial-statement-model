# Financial Statement Model 📊

This project is a three-statement financial model built in Excel.

## 🔍 Overview
The model includes:
- Income Statement
- Balance Sheet
- Cash Flow Statement

All statements are dynamically linked and balanced.

## 📈 Key Features
- Fully integrated three-statement model
- Forecasting from 2025–2029
- Working capital schedule
- Debt and interest schedule
- Depreciation and capex modeling
- Balance sheet validation check

## 🧠 Financial Insights
The model demonstrates:
- Revenue growth forecasting
- Cost structure analysis
- Profitability trends
- Cash flow generation
- Financial position over time

## 🛠 Tools Used
- Microsoft Excel

## 📂 File
- Three Statement Model.xlsm

## 🎯 Purpose
This project showcases practical financial modeling and analysis skills used in real-world finance roles.
